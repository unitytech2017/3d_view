# 3D 파일 뷰어

웹 브라우저에서 다양한 3D 모델 파일을 실시간으로 확인할 수 있는 뷰어입니다.

## 🎯 주요 기능

- **다양한 파일 형식 지원**: OBJ, FBX, GLTF, GLB, STL, USDZ, PLY, 3DS
- **실시간 3D 렌더링**: Three.js 기반 고품질 렌더링
- **인터랙티브 컨트롤**: 마우스로 회전, 확대/축소, 이동
- **커스터마이징**: 배경색, 조명, 와이어프레임 모드 등
- **애니메이션 지원**: FBX, GLTF 파일의 애니메이션 재생
- **스크린샷 저장**: 현재 뷰를 PNG로 다운로드

## 🚀 사용 방법

1. 파일 업로드 영역에 3D 모델 파일을 드래그 앤 드롭하거나 클릭하여 선택
2. 마우스로 모델을 회전, 확대/축소하여 확인
3. 좌측 패널에서 다양한 뷰어 옵션 조절

## 🎨 뷰어 설정

- **배경 색상**: 뷰어 배경색 변경
- **조명 강도**: 모델 조명 밝기 조절
- **와이어프레임 모드**: 모델의 메쉬 구조 확인
- **자동 회전**: 모델 자동 회전 활성화
- **애니메이션 재생**: 애니메이션이 포함된 모델의 재생 제어

## 💻 기술 스택

- Three.js (r128)
- HTML5 / CSS3
- JavaScript
